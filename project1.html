Part 2: Calculate Statistics with Pandas#

Now letâ€™s calculate the three measures of central tendency:

  • Mean: The average rating

  • Median: The middle value when sorted

  • Mode: The most frequent rating

# Calculate statistics using pandas
mean_pandas = selected_column.mean()
median_pandas = selected_column.median()
mode_result = selected_column.mode()

# Handle mode (might have multiple modes)
if len(mode_result) > 0:
    mode_pandas = mode_result[0]  # take first mode
else:
    mode_pandas = None

# Display results
print("Statistics from Pandas:")
print(f"Mean:   {mean_pandas:.4f}")
print(f"Median: {median_pandas:.4f}")
print(f"Mode:   {mode_pandas}")

# Let's see what these numbers tell us
print(f"\nInterpretation:")
print(f"The average rating of {mean_pandas:.2f} suggests...")
if mean_pandas > 4:
    print("  - Investors have a positive view of Mutual Funds")
else:
    print("  - Investors are neutral/cautious about Mutual Funds")

Part 3: Manual Calculations (The Hard Way)#

Now letâ€™s calculate the same statistics WITHOUT using pandas or any statistics library. This helps us understand whatâ€™s happening behind the scenes.

# Read data manually
import csv

data = []
with open("Finance_Trends.csv", "r") as file:
    csv_reader = csv.DictReader(file)
    for row in csv_reader:
        data.append(int(row["Mutual_Funds"]))

print(f"Read {len(data)} values")

Read 12000 values

# Mean = sum / count
mean_manual = sum(data) / len(data)
print(f"Mean: {mean_manual:.4f}")

Mean: 2.5346

# Median = middle value
sorted_data = sorted(data)
n = len(sorted_data)

if n % 2 == 1:
    median_manual = sorted_data[n // 2]
else:
    median_manual = (sorted_data[n // 2 - 1] + sorted_data[n // 2]) / 2

print(f"Median: {median_manual:.4f}")

Median: 2.0000

# Mode = most frequent value (MUST USE DICTIONARY!)
counts = {}
for value in data:
    if value in counts:
        counts[value] += 1
    else:
        counts[value] = 1

# Find most common
mode_manual = max(counts, key=counts.get)
print(f"Mode: {mode_manual}")

Mode: 2

Conclusion#

Through this analysis of 12,000+ investment preferences, we discovered:

  • The average Mutual Fund rating is 2.5346, which is below the midpoint of 4, suggesting investors are somewhat skeptical of Mutual Funds.

  • The median of 2.0000 confirms that more than half of investors gave ratings of 2 or below, showing a general lack of enthusiasm.

  • The mode of 2 (appearing 6343 times) indicates this is the most common sentiment - mild interest but not strong preference.

This tells us that investors generally view Mutual Funds as a lower-priority investment option compared to other vehicles.
